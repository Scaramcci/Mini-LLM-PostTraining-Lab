# Mini-LLM-PostTraining-Lab

This is my first long-term LLM learning project.

## Goal

Build a small but complete LLM post-training pipeline:

1. Train a mini GPT from scratch
2. Build tokenizer and data pipeline
3. Run Qwen LoRA SFT
4. Run DPO preference alignment
5. Try GRPO with verifiable rewards
6. Serve models with vLLM
7. Build a tool-use agent
8. Evaluate and write a final report
