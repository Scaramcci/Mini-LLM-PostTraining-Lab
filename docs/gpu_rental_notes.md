# GPU Rental Notes

这个项目预计用一个月左右推进，所以不建议一开始月租 GPU。先本地学习和写代码，真正训练时按小时租。

## Rule

每次开 GPU 前先写清楚：

- 今天跑什么？
- 需要哪个模型？
- 需要哪个数据？
- 命令是什么？
- 预计多久？
- 结果保存到哪里？

## Session Template

```markdown
## 2026-xx-xx GPU Session

- Provider:
- GPU:
- Price:
- Start time:
- End time:
- Stage:
- Goal:
- Model:
- Dataset:
- Command:
- Output path:
- Result:
- Cost estimate:
- Mistakes / notes:
```

## Stage 01 Advice

Stage 01 优先本地跑。如果本地太慢，再租 4090 跑 1-3 小时即可。
